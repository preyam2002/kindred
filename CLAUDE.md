# Kindred

## Overview
Web app with 3D visualization, AI integration, and Supabase auth.

## Commands
```bash
npm run dev       # Next.js dev
npm run build     # Production build
```

## Tech
- Next.js 16, React 19, TypeScript
- Three.js / React-Three-Fiber (3D)
- Anthropic SDK (AI)
- Supabase (DB + Auth via Next-Auth)
- Tailwind CSS
